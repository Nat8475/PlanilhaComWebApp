🔨 Manutenção Rápida do Sistema

Realiza pequenos ajustes no sistema sem recriar ou apagar nenhuma aba ou dado:

✅ Reaplica as cores de status em todas as abas
✅ Atualiza as métricas do Dashboard
✅ Verifica e reinstala os triggers automáticos
✅ Inclui as abas extras de fornecedores criadas pelo usuário

Nenhuma aba é recriada ou limpa. Use esta opção para corrigir cores incorretas, contadores desatualizados ou triggers parados — sem risco de perder lançamentos.
